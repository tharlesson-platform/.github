# Tharlesson Platform

Este espaço centraliza os projetos de plataforma, SRE, DevOps e GitOps que construí.

| `terraform-modules` | Terraform AWS Platform Blueprint | Módulos reutilizáveis e stacks multi-conta/multi-ambiente. | [tharlesson/terraform-modules](https://github.com/tharlesson/terraform-modules) |
